# Calculator Website

A simple calculator application built using HTML, CSS, and JavaScript.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by Zero Validation
* User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Use

1. Enter two numbers.
2. Click any arithmetic operation button.
3. View the result instantly.
